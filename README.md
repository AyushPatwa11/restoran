# Restoran — Bootstrap Restaurant Template

This repository contains a static Bootstrap-based restaurant website template (HTML, CSS, JS, and assets).

## Project Structure

- `index.html` — Home page
- `about.html`, `menu.html`, `service.html`, `contact.html`, etc. — Site pages
- `css/` — Compiled CSS (`bootstrap.min.css`, `style.css`)
- `js/` — Custom JavaScript (`main.js`)
- `img/` — Images used by the template
- `lib/` and `scss/` — Third-party libraries and SCSS sources

## Usage

1. Clone the repository:

   git clone https://github.com/AyushPatwa11/restoran.git

2. Open `index.html` in a browser to view the template locally.

No build step is required — this is a static template.

## Customization

- Edit HTML files to update content.
- Modify `css/style.css` or the SCSS sources in `scss/` for styling changes.
- Replace images in `img/` with your own assets.
